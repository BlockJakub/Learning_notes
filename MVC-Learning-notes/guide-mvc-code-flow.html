Practical MVC: Registration Example

How the Controller and Model work together using PHP.

Integrating View, Controller, and Model

This simplified example assumes separate **Controller** and **Model** files. The **View** is the HTML form the user submits.
Controller: `UserController.php`

Handles input, delegates tasks, returns response.

<?php
require_once 'UserModel.php';
require_once 'EmailService.php';

class UserController {
    private $userModel;
    private $emailService;

    public function __construct() {
        // Dependencies are initialized here
        $this->userModel = new UserModel();
        $this->emailService = new EmailService();
    }

    // <-- 1. Controller Receives Request -->
    public function register($formData) {
        $email = $formData['email'] ?? null;
        $password = $formData['password'] ?? null;

        // Basic validation (more required in real life)
        if (empty($email) || empty($password)) {
            return ["status" => "error", "message" => "Missing fields"];
        }

        // <-- 2. Controller Delegates to Model -->
        $userId = $this->userModel->createUser($email, $password);

        if ($userId) {
            // <-- 3. Controller Delegates Token/Email Generation -->
            $token = $this->userModel->generateVerificationToken($userId);
            
            $link = "https://yourdomain.com/verify?token=" . $token;
            
            $this->emailService->send(
                $email, 
                "Verify Your Account", 
                "Click here to verify: " . $link
            );

            // <-- 4. Controller Returns Status -->
            return ["status" => "success", "message" => "Check your email."];
        }

        return ["status" => "error", "message" => "User already exists."];
    }
}
// Imagine a router calls: (new UserController())->register($_POST);
?>

The Connection (The Router)

The code above shows the classes, but the missing piece is the **Router**. This is what links the URL request to the Controller method.

View (HTML Form) $\rightarrow$ `POST /register` $\rightarrow$ Router $\rightarrow$ `UserController->register()` $\rightarrow$ Model/Service.
